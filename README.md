# What You Wear — interaktiver Webprototyp

Der Prototyp besteht aus einer ausführlichen, faktenbasierten Landingpage und einem bedienbaren B2B-Dashboard. Er kommt ohne Build-Schritt und ohne externe Bibliotheken aus.

## Lokal starten

Im Projektordner einen lokalen Webserver starten:

```bash
python3 -m http.server 4173
```

Danach `http://localhost:4173` im Browser öffnen.

## Enthaltene Funktionen

- lange responsive Landingpage mit Konzeptbildern und Primärquellen
- interaktives Scan-Beispiel und Score-Erklärung
- eigenständige mobile B2C-Produktseite für QR-Links mit aufklappbaren Score-Details
- Dashboard mit Übersicht, Produktsuche und Filtern
- Produktanlage mit lokal gespeicherten Demo-Daten
- Live-Berechnung für Material- und Produktionsscore
- Zertifikat-Upload als lokale Demo
- editierbare Lieferkettenstufen und Produktpass-Status
- Analytics-Ansichten
- Exporte als CSV, JSON, HTML-Bericht und SVG-Label

Die Daten werden im Browser gespeichert. Das Projekt ist ein Hochschulprototyp und keine akkreditierte Zertifizierung, Ökobilanz oder Rechtsberatung.
